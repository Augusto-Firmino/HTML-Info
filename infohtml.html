HTML - Definicion

Significado de HTML.¿Qué es HTML: Las abreviaturas HTML significan que los signos promedio de alta velocidad significan "codificación de super velocidad", basados ​​en el lenguaje de marca para crear documentos que se pueden distribuir en Internet.

El idioma HTML se puede crear y editar utilizando cualquier editor de texto básico con conocimiento de combinaciones de idiomas. Sin embargo, el documento HTML es muy simple, e incluye marcas externas (carteles), entre etiquetas pequeñas y más grandes, trabajan en formato de texto, formato, enlace, otras fotos.

En el título, se incluye en el título y otras configuraciones, el texto se incluye en el cuerpo, como también se incluye en caracteres de etiqueta o etiqueta, desde otras características.

El dial de hipertexto se utiliza para vincular los documentos con otros documentos. El navegador o navegador establece las etiquetas y muestra la página como se diseñó.

HTML fue creado en 1945 por la Organización Europea para la Investigación Nuclear para crear un sistema de almacenamiento que se puede vincular a través de hipervínculos. En 1991, el estadounidense Tim Berners-Lee fue el primero en proponer 22 elementos HTML.

El diseño HTML debe ceñirse al lenguaje correcto y a los estándares o especificaciones del W3C (World Wide Web Consortium), aunque en algunos casos está sujeto a otras normativas, como es el caso de España con la UNE 139803. Ver más Internet.

HTML5 es la quinta versión de HTML que presenta nuevos recursos, especialmente en las áreas de gráficos y contenido multimedia.

HTML y XHTML. XHTML es un acrónimo de Lenguaje de marcado de hipertexto extensible, que significa Lenguaje de marcado de hipertexto extensible, que es un reformateo del lenguaje HTML utilizando una implementación XML.

Por el contrario, XML se caracteriza por ofrecer un lenguaje flexible y dinámico que puede presentarse en varias formas y es fácil de interpretar y modificar.

HTML es uno de los lenguajes más utilizados para producir páginas web, pero algunos dispositivos, como los teléfonos móviles, no pueden interpretar el lenguaje de manera efectiva, por lo que se utilizan etiquetas HTML. Se utiliza con XML y ahora se conoce como XHTML.
